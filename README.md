1. Последовательный запрос данных (день, месяц, год) с обработкой исключений (ValueError) на случай некорректного ввода.
2. Определение дня недели с использованием модуля datetime. 
3. Определение високосного года с помощью встроенного модуля calendar (calendar.isleap).
4. Расчет точного возраста с учетом того, прошел ли день рождения в текущем году. 
5. Вывод даты рождения на "электронное табло". Для этого был создан словарь (Dictionary), где ключами выступают строковые представления цифр от '0' до '9', а значениями — массивы строк, формирующие символы из звездочек. Функция перебирает отформатированную дату (ДД ММ ГГГГ) и выводит соответствующие элементы массивов построчно. 
  *Добавлен цикл while True, который не пропустит пользователя дальше, пока он не введет реально существующую дату (учитывая количество дней в конкретном месяце високосного года) 
  **get_age_word (теперь работает корректно) проверяет окончания чисел (исключая числа от 11 до 14, которые всегда «лет», а для остальных смотрит на последнюю цифру).
